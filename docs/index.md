# ChapterForge Documentation

ChapterForge is a Windows application that combines MP3 files into a single master MP3 with embedded chapter markers.

## Getting Started

1. Download ChapterForge from the [releases page](https://github.com/BITS-ACB/chapterforge/releases)
2. Run the installer
3. Open a folder of MP3 files
4. Build your master audiobook

## Features

- **Combine MP3 files** - Turn a folder of MP3s into one master with chapters
- **Fully accessible** - Keyboard-only operation, screen reader compatible
- **Beautiful UI** - Light and dark themes, customizable text size
- **Multiple formats** - MP3 with ID3v2 chapters or M4B audiobooks
- **Auto-building** - Watch folders and automatically build new audiobooks
- **Job files** - Save and reuse your setups
