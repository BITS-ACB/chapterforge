What ChapterForge Does

Build a Chaptered Audiobook

Combine a folder of MP3s into one master file with a chapter for each track. Open the finished file in any podcast or audiobook app and navigate chapter by chapter.

Works With Your Favorite Apps

Chapters are recognized by Apple Podcasts, Overcast, Pocket Casts, AntennaPod, VLC, foobar2000, AIMP, and any other app that supports standard chapter formats.

Edit Before You Build

Chapter titles are filled in from your file names. Adjust them, reorder tracks, and set the book's title, author, cover image, genre, and year before building.

Fix Existing Files

Open a finished chaptered MP3 or M4B to correct its chapters and tags without rebuilding from scratch. Merge chapters, split at a point, or fix a title in place.

Find Chapters Automatically

Have one long recording instead of separate tracks? ChapterForge can detect natural chapter breaks by finding silence in the audio.

Watch Folders Automatically

Set ChapterForge to watch a folder and it will build a new book automatically every time you drop in a set of MP3s. You get a Windows notification when it finishes.

Built for Accessibility

Screen Reader Support

NVDA, JAWS, and Narrator all announce controls, status updates, and build progress clearly. No workarounds needed.

Keyboard Navigation

Every feature is reachable by keyboard. Every button, checkbox, and list has a mnemonic and a visible label.

High-Contrast Theme

An optional high-contrast theme and adjustable text size are available in Tools > Settings.

Non-Blocking Builds

The app stays responsive while building. Progress is announced as it happens; you can cancel at any time.

Windows 10 and 11

Runs on Windows 10 and 11 (64-bit). No additional software required - FFmpeg and everything else is bundled.
