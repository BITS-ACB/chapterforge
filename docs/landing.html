How it works

Three steps. No technical knowledge required.

1

Open a folder

Point ChapterForge at any folder of MP3 files. Chapters are created automatically, one per track, sorted and titled from your file names.

2

Refine and tag

Edit chapter titles, reorder tracks, add cover art, and fill in the author and album. Most fields are pre-filled from your folder and file names.

3

Build and preview

Press Build. ChapterForge assembles everything into one master file. Listen in the built-in player before you share it.

Everything you need to build great audio

ChapterForge covers the full workflow from raw files to finished, distributable audiobook.

Chapters that work everywhere

Your finished file plays in Apple Podcasts, Overcast, Pocket Casts, AntennaPod, VLC, foobar2000, AIMP, and any other app that reads standard ID3v2 chapter markers. One build, universal compatibility. Export as MP3 with chapter markers or M4B audiobook format - whichever your audience prefers.

Edit, merge, and split chapters

Titles come from your file names, already meaningful before you start. Reorder tracks with a keystroke. Merge two short chapters into one. Split a chapter at any playback position while listening in the built-in player. Rename in bulk with a single dialog. Every change is live before a single file is written. Full chapter editing guide.

Hear it before you share it

ChapterForge includes a fully keyboard-accessible audio player so you can review your work inside the app. Play from any chapter, skip between chapters, rewind or jump forward in configurable steps, and adjust volume - all without leaving ChapterForge. Player controls and shortcuts.

Find chapters in one long recording

Don't have separate tracks? ChapterForge can listen for natural silence in a long recording and mark chapter breaks automatically. Review the results, adjust any boundary that needs it, then build. Perfect for digitized tapes, recorded lectures, or radio broadcasts. About auto-chapter by silence.

Fix existing chaptered files

Open a finished MP3 or M4B to correct a chapter title, adjust where a chapter starts, replace the cover art, or update any tag - without touching the audio. Merge adjacent chapters, split one at any point, or trim the beginning and end. Save in place or Save As a new file and keep your original intact.

Watch folders, build automatically

Tell ChapterForge to watch a folder and walk away. When you drop in a set of MP3s, ChapterForge detects them, waits for the files to settle, and builds the book on its own - then notifies you with a Windows toast and a screen-reader announcement. Set up the background watcher. Enable start at sign-in and your audiobook folder runs itself.

Rich tags and automatic cover art

Set the title, artist, album, album artist, genre, year, comment, and track number on your master file. ChapterForge scans your source folder for a cover image and picks it up automatically - or choose one manually. Tags are written as ID3v2.3 for the widest player compatibility.

Import and export chapter lists

Already have chapter markers defined elsewhere? Import from Audacity label files, CUE sheets, or plain timestamp text. Export in the same formats, or as a Podcasting 2.0 JSON sidecar with optional per-chapter URLs and images. Chapter lists move freely in and out.

Built for accessibility, first

ChapterForge is made by Blind Information Technology Solutions (BITS), a team that builds software for blind and low-vision users first, and everyone else second. That shows in every detail.

Screen reader support

NVDA, JAWS, and Narrator all announce controls, status, and build progress clearly. No workarounds, no silent buttons, no mystery. Screen reader details.

Full keyboard navigation

Every feature is reachable by keyboard. Every button, checkbox, and list has a mnemonic and a visible label. Full keyboard shortcut reference.

High-contrast theme

An optional high-contrast theme and adjustable text size (8pt to 24pt) are available in Tools > Settings for comfortable use in any lighting condition.

Non-blocking builds

Long operations run in the background. The interface stays alive, progress is announced as it happens, and you can cancel at any time without leaving a broken file behind.

Guided startup wizard

First launch opens a step-by-step wizard that introduces every major feature, configures defaults, and gets you to your first build without reading a manual.

Help when you need it

Every control has a tooltip. The full User Guide is bundled with the app and opens in your browser. Save Diagnostics packages a support report in one click.
