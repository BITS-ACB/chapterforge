Powerful Features

🎵

Multi-Format Support

Import from MP3, FLAC, WAV, AAC, M4A, OGG, WMA, OPUS, and MP2. Export to MP3 with ID3v2 chapters or M4B audiobooks.

♿

Full Accessibility

Keyboard-only operation, comprehensive screen reader support (NVDA, JAWS, Narrator), and customizable high-contrast themes.

⚡

Intelligent Detection

Automatically detect chapter breaks in long audio recordings by analyzing silence gaps with configurable precision.

🎯

Professional Tagging

Comprehensive ID3 tag editor with custom field support and automatic album art detection and embedding.

📦

Batch Processing

Convert entire libraries with a single setup. Process multiple folders automatically with consistent tagging.

💾

Job File System

Human-readable text files store complete configurations. Save, reuse, and version control your setups.

👁️

Background Processing

System tray watcher automatically processes new audio folders with automatic stabilization detection.

⌨️

Command-Line Interface

Full CLI access for automation and scripting workflows. All graphical features available from the command line.
