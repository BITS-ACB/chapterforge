# FOR IMMEDIATE RELEASE

## ChapterForge 2.0.0 Launches as the Premier Accessible Audio Chapter Management Solution

### Revolutionary Software Delivers Unprecedented Accessibility, Performance, and Professional-Grade Tools for Audiobook Creators, Podcasters, and Visually Impaired Users

**Tucson, AZ — June 5, 2026** — Blind Information Technology Solutions (BITS), a leader in accessible technology solutions, today announced the official launch of ChapterForge 2.0.0, their acclaimed audio chapter management software. This inaugural release introduces universal format compatibility, a completely accessible and modern interface, and comprehensive accessibility features that set new standards for professional audio production.

ChapterForge 2.0.0 transforms the landscape of audio chapter management by combining powerful editing capabilities with unparalleled accessibility features. The software enables users to effortlessly convert collections of audio files into professionally chaptered masterpieces, whether for audiobooks, podcasts, music albums, or educational content.

"This release represents the culmination of years of development focused on accessibility and user experience," said Jeff Bishop, President of BITS. "We've created a tool that ensures everyone, regardless of ability, can create professional-quality audio content independently while maintaining the highest standards of quality and performance."

### Unmatched Accessibility for All Users

Built from the ground up with full keyboard navigation, comprehensive screen reader support, and customizable accessibility features, ChapterForge 2.0.0 makes professional audio editing possible for everyone. The software includes:

- Full keyboard control with customizable shortcuts
- Intelligent screen reader announcements with context-aware feedback
- High-contrast themes and scalable interface elements
- Voice-guided workflow for hands-free operation
- Compatibility with NVDA, JAWS, and Windows Narrator

### Universal Format Support and Professional Features

ChapterForge 2.0.0 breaks down format barriers by supporting import from and export to virtually any audio format, including MP3, FLAC, WAV, AAC, M4A, OGG, and WMA. Professional-grade features include:

- Multi-threaded processing engine delivering up to 5x faster builds
- Comprehensive ID3 tag editor with custom field support
- Album art embedding with automatic cover art detection
- Batch processing wizard for library-wide conversions
- Silence-based chapter detection for single long recordings
- Lossless audio processing when possible
- Advanced audio trimming and cutting tools
- Chapter splitting functionality for long recordings
- Background folder watching with system tray integration
- Job file system for saving and reusing configurations
- Command-line interface for automation and scripting

### Enhanced User Experience with Modern Interface

The designed interface features adaptive layouts, smooth animations, and a sleek dark/light theme toggle. A new intuitive drag-and-drop chapter reordering system which is also fully keyboard friendly, combined with precision timeline editing tools, makes complex editing tasks simple and efficient.

### Official Launch Marks Milestone in Accessible Technology

As the first official release of ChapterForge, version 2.0.0 represents a significant milestone in accessible audio software development. The software incorporates feedback from beta testers and accessibility advocates to deliver a robust, professional tool that serves the needs of both casual users and content creators.

### Revolutionary Features in ChapterForge 2.0.0

#### Multi-Format Audio Processing
Support for virtually any audio format including MP3, FLAC, WAV, AAC, M4A, OGG, WMA, OPUS, and MP2. Import from any supported format and export to MP3 with ID3v2 CHAP/CTOC chapters, M4B/AAC with native MP4 chapters, or FLAC lossless format.

#### Intelligent Silence-Based Chapter Detection
Automatically detect chapter breaks in long audio recordings by analyzing silence gaps, making it easy to convert single long recordings into properly chaptered audiobooks. Configurable noise threshold and minimum silence duration settings allow for precise control.

#### Advanced Chapter Management
- Drag-and-drop chapter reordering with full keyboard accessibility
- Precision timeline editing tools for exact chapter boundary placement
- Batch editing capabilities for efficient workflow
- Chapter merging and splitting functionality
- Chapter title customization with pattern-based renaming

#### Audio Trimming and Cutting Tools
- Lossless audio trimming with precise start/end point selection
- Pre-listen functionality to hear edits in context
- Save trimmed selections as separate files using lossless FFmpeg copy
- Split one long recording into individual chapter files

#### Background Processing and Automation
- System tray background watcher that automatically processes new audio folders
- Configurable watch processes with custom naming templates
- Automatic stabilization detection to prevent processing incomplete copies
- Success/failure notifications with detailed reporting

#### Job File System
- Human-readable text files that store complete project configurations
- Save and reuse complex configurations with editable text files
- Version control friendly for collaborative workflows
- Template system for common project types

#### Professional Tagging System
- Comprehensive ID3 tag editor with support for all standard fields
- Custom field support for specialized metadata
- Album art embedding with automatic detection from folder contents
- Integration with MusicBrainz and Discogs databases (manual lookup)

#### Batch Processing Wizard
- Convert entire libraries of audiobooks with a single setup
- Process multiple sub-folders automatically with consistent tagging
- Automatic folder detection and processing
- Progress tracking and detailed reporting

#### Command-Line Interface
- Full CLI access with detailed progress reporting
- Support for automation and scripting workflows
- All graphical features available through command line
- Dry-run mode for testing configurations

#### Real-time Audio Visualization
- Waveform display with zoom and navigation controls
- Real-time playback preview with instant chapter navigation
- Audio quality analysis and issue detection
- Spectral analysis and frequency visualization

#### Lossless Processing Options
- Lossless concatenation when all files share the same format
- Automatic clean re-encode when formats differ
- Per-file normalization options with LUFS targeting
- Fade-in/fade-out effects for smooth transitions

#### Advanced Export Options
- Configurable bitrate settings for different quality requirements
- Inter-chapter gap insertion with precise timing control
- Podcasting 2.0 chapters JSON sidecar file generation
- Multiple export format support (MP3, M4B, FLAC)

### Technical Specifications

- **Platform**: Windows 10/11 (64-bit)
- **Accessibility**: Full keyboard navigation, screen reader support (NVDA, JAWS, Narrator)
- **Formats**: Input - MP3, FLAC, WAV, AAC, M4A, OGG, WMA, OPUS, MP2; Output - MP3, M4B, FLAC
- **Dependencies**: Bundled FFmpeg for all audio processing
- **Performance**: Multi-threaded processing engine with up to 5x faster builds
- **Licensing**: MIT License - free for personal and commercial use
